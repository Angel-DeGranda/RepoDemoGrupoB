07 — Listas

Clase .list-reset para quitar estilos por defecto

Vista previa:

Lista normal (sin clase):

  • Elemento uno
  • Elemento dos
  • Elemento tres

Con .list-reset (sin bullets ni padding):

  • Elemento limpio
  • Elemento limpio
  • Elemento limpio
Código:
<ul class="list-reset"> <li class="mb1 py1 border-bottom">Elemento limpio</li> <li class="mb1 py1 border-bottom">Elemento limpio</li> <li class="mb1 py1">Elemento limpio</li> </ul>
